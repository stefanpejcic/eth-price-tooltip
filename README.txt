== Description ==

Plugin will find mentions of Ethereum in your texts and automatically add a tooltip to it with actual price in USD and EUR. No need to do any setting.

![screenshot](https://plugins.club/wp-content/uploads/2023/01/chrome_pWlFzBqkI6.png)

For Bitcoin please use [Bitcoin price tooltip](https://wordpress.org/plugins/bitcoin-price-tooltip/)
